# 미샵 템플릿 OS

상세페이지 자동화를 위한 이미지 편집 템플릿 제공 프로그램입니다.

## 핵심 기능
- 상단 탭 UI 기반 템플릿 생성 / 관리 / 활용
- 참조 JPG 업로드 또는 예시 디자인 불러오기
- AI 기반 이미지 영역 / 카피 영역 자동 추천
- 드래그 박스 추가로 수동 보정
- 미샵 전용 기본 블록 프리셋(3초 훅 / 대표 이미지 / USP / 디테일 컷)
- 템플릿 JSON 저장 및 재사용
- 즉시 사용용 JPG 출력
- 포토샵 후편집용 JSX + assets 패키지 ZIP 출력

## 실행 방법
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 출력물 구성
- `rendered.jpg` : 즉시 사용 가능 이미지
- `misharp_template.jsx` : 포토샵 스크립트
- `assets/` : 포토샵에서 불러올 이미지 파일
- `template.json` : 템플릿 구조 원본

## 주의
- 이번 버전은 직접 PSD 바이너리를 쓰는 방식이 아니라, **포토샵에서 후편집 가능한 JSX 패키지 방식**입니다.
- 폰트 변경/세밀한 타이포 조정은 포토샵에서 마무리하는 실무형 구조입니다.
