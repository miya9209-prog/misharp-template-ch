# 미샵 템플릿 OS

상세페이지 상단형 디자인을 템플릿으로 등록하고, 등록된 템플릿에 이미지와 카피를 입력해 JPG와 포토샵용 JSX 패키지로 내보내는 Streamlit 프로그램입니다.

## 핵심 구조
- **템플릿 생성부**: 참조 JPG 기반으로 이미지 슬롯 / 카피 박스를 정의하고 템플릿 저장
- **템플릿 활용부**: 저장된 템플릿 선택 → 이미지 업로드 + 카피 입력 → 결과 JPG 출력
- **포토샵 연동**: JSX 파일을 함께 내려받아 포토샵에서 열고 최종 PSD로 마무리 가능
- **상단 탭 UI**: 향후 셀러OS 편입을 고려하여 좌측 메뉴 대신 상단 탭 구조 사용

## 실행 방법
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 폴더 구조
- `templates/` : 저장된 템플릿 JSON
- `examples/` : 참조 이미지 및 예시 이미지
- `outputs/` : 필요 시 확장용 출력 폴더

## 현재 버전의 범위
현재 버전은 **실행 가능한 1차 베타 레포**입니다.

직접 PSD 바이너리를 생성하는 대신, 실무에서 더 안정적으로 활용할 수 있도록 다음 방식으로 구성했습니다.

1. JPG 결과물 즉시 다운로드
2. 템플릿 JSON 저장
3. 포토샵용 JSX 스크립트 다운로드
4. 업로드한 소스 이미지와 함께 ZIP 패키지 제공

포토샵에서 JSX를 실행한 뒤 레이어 위치/서체를 미세 조정하고 PSD로 저장하면 됩니다.
